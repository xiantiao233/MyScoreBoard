# MyScoreBoard
让玩家设计自己的 计分板 吧！

---

> 指令
- /mysb reload
  - 权限: OP | myscoreboard.reload
  - 重载创建配置文件 (不建议使用,请重启服务器重载)
- /mysb set <add|set|remove|resetting> [行]
  - 权限: OP | myscoreboard.use
    - add - 向计分板增加内容
    - set - 设置某一行的文字
    - remove - 移除某一行
    - resetting -清空
- /mysb look
  - 看自己的计分板文字

> 使用 `!:`-表示空格 `%player%`-表示玩家名字  
> 支持 PlaceholderAPI
